# Questão 3 — Perguntas → Tipo de gráfico
# =========================================

1. Qual canal gera mais receita? → Gráfico de barras  
2. Como a receita evolui ao longo do tempo? → Gráfico de linha  
3. A margem varia entre categorias? → Boxplot  
4. Receita alta significa lucro alto? → Scatter plot

# =========================================
# Questão 8 — Insight Canal
# =========================================

O canal Online é o que gera mais receita, seguido pelo Marketplace.  

Os canais Loja Física e Televendas ficam atrás em volume.  

Uma ação estratégica seria investir mais no canal Online, já que ele lidera o faturamento.

# =========================================

# =========================================
# Questão 18 — Comunicação vs Decoração
# =========================================

Um bom gráfico deve responder uma pergunta de negócio de forma clara.

O foco deve ser sempre facilitar a tomada de decisão.

# Questão 20 — Comparabilidade e honestidade
# =========================================

Ordenar gráficos melhora a leitura e comparação entre categorias.  

Manipular eixos pode distorcer a percepção e levar a decisões erradas.  

A visualização deve ser fiel aos dados.
